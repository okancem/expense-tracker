# Expense Tracker

A full-stack expense tracking application built with the **MERN stack**. Users can securely manage their income and expenses, view summaries, and analyze their financial habits.

## Features
- User authentication with JWT
- Add, update, and delete income/expense transactions
- Categorize expenses
- View real-time financial summaries
- Responsive and user-friendly UI
- API testing with Postman

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Testing:** Postman (optional: Jest for unit testing)
